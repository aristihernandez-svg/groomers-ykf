What is Skycare YKF

Skycare YKF is a mobile-first operations platform for the ground crew team at CYKF. It replaces paper checklists, group texts, and spreadsheets with a single app that every team member runs on their phone — no installation required. Built as a Progressive Web App, it works offline, sends push notifications, and tracks everything from daily cleaning tasks to live aircraft positions.

🛠 The Problem It Solves

Ground crews at regional airports coordinate across multiple roles — groomers, mechanics, pilots, buyers — using informal tools that create gaps: missed tasks, forgotten supplies, unknown aircraft ETAs, and no paper trail. Skycare centralizes all of this into one real-time shared system accessible from any phone.

  • High turnover, solved by design.

    Groomers turn over every 6 to 12 months. Skycare embeds the institutional knowledge into the platform — a new hire on day one follows the same process as someone who has been here for years, because the procedures live in the app, not in someone's memory.

  • Uniform work across the team.

    Every groomer sees the same task lists, the same procedures, the same checklists — regardless of experience level. The quality of a shift no longer depends on who's working it.

  • A foundation for multi-base consistency.

    Because the platform is cloud-based and configurable, the same processes deployed at CYKF can be replicated at any Skycare base — creating company-wide operational standards without rebuilding anything from scratch.

  • More than a task tool — a management layer.

    Skycare is not just about cleaning planes. It gives supervisors and management a live operational picture — task completion, fleet positions, vehicle status, supply levels, shift handoffs — all in one place, updated in real time. The team executes; the platform reports. Management stays informed without having to ask.

Creative Process
Phase 1 — Foundation
  • Daily task checklist for groomers
  • Task data seeded from Google Sheets, migrated to Firestore
  • PWA shell, service worker, offline support
  • 4-theme UI system
Phase 2 — Crew Operations
  • Crew car tracking — fuel gauges, odometer, QR codes per vehicle
  • Monthly audit system with photo capture
  • Shopping list with per-store filtering and shared cart
  • SOPs and procedure cards
Phase 3 — Notifications
  • Web Push notifications via VAPID — works on iOS, Android, desktop
  • GitHub Actions serverless backend for all notification delivery
  • Coffee reminders, evening lockup, fuel alerts, MX parts queue
Phase 4 — Fleet Intelligence
  • Live map tracking all 10 fleet aircraft in real time
  • Distance, altitude, speed, and ETA per aircraft
  • 60 nm arrival alert with 2-hour cooldown per tail
  • Position data cached to Firestore — no direct API calls from the browser

Push Notification System
🔔 How It Works
  • User taps the bell icon → browser generates a push subscription and saves it to Firestore
  • GitHub Actions scripts read all subscriptions and deliver pushes via the web-push library
  • The service worker on each device receives the push and shows the native OS notification
  • Works on iOS 16.4+, Android, and desktop
  • Expired subscriptions are automatically removed after each send batch

Business Case

$0/month infrastructure cost at current scale:

  • GitHub Pages — hosting
  • Firestore Spark — database
  • GitHub Actions on public repo — serverless backend
  • cron-job.org — scheduled triggers

The app runs in production at zero operating cost. The first paying customer is 100% margin above the cost of building it.

📈 Expansion Path

The platform is not aviation-specific. The core loop applies to any team managing tasks, vehicles, supplies, and shift handoffs:

  • Regional aviation teams and FBOs
  • Hotel housekeeping operations
  • Airport ground handlers
  • Facilities and fleet management

Skycare YKF is the proof of concept. The product is the platform.
