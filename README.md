# Groomers YKF

Aviation Grooming Team — YKF Operations PWA

## Setup

1. Open `index.html` and replace `YOUR_SHEET_ID_HERE` with your Google Sheet ID
2. The Sheet ID is the long string in your sheet URL:
   `https://docs.google.com/spreadsheets/d/**SHEET_ID_HERE**/edit`
3. Make sure your Google Sheet is shared as "Anyone with the link can view"

## Files

- `index.html` — Main PWA app
- `manifest.json` — PWA manifest (name, icon, colors)
- `sw.js` — Service worker (offline support)
- `icon-192.svg` — App icon (small)
- `icon-512.svg` — App icon (large)
